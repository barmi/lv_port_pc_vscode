# 프로젝트 문서 가이드

이 디렉토리는 프로젝트의 설정 및 개발 도구 사용법에 대한 문서들을 포함하고 있습니다.

## 문서 목록

1.  **[Python Tool Guide](tool_guide.md)**
    *   `tool` 디렉토리에 있는 각종 Python 스크립트들의 역할과 사용법 (폰트 변환, 아이콘 처리 등)을 설명합니다.
2.  **[Hardware Setting Guide](hw_setting.md)**
    *   하드웨어 설정 및 관련 구성에 대해 설명합니다.

## 개발 참고 사항
*   이미지나 폰트를 수정할 때는 `tool_guide.md`를 참고하여 관련 스크립트를 실행해 주시기 바랍니다.
*   문서의 이미지들은 `img/` 디렉토리에 저장되어 있습니다.
